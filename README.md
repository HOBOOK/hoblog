### Configuration

Add backend/config/default.json

```
{
    "servicePort" : 3000
    , "dbHost" : {your db host} ex)"127.0.0.1"
    , "dbPort" : {your db port} ex)27017 
    , "dbName" : {your db name} ex)"leaflog"
    , "dbAuthenticate" : true
    , "dbUser" : {your db user} ex)"admin"
    , "dbPass" : {your db password} ex)"1234"
}
```
